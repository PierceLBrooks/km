Superconcept: Be-Inaccessible

Specification for 'Be-Obstructed'[view kml]

Concept: Be-Obstructed

Definition: An object becomes restricted in its movement. An object is put in this state by Obstruct. An object is taken out of this state by Unobstruct.

Required Relationships:

object: the thing that is obstructed

Examples:

  1. The goalie blocked the attacker at every turn.
    object: the attacker
    agent: the goalie
    2. Comment: Be-Blocked should not be used because Be-Blocked only applies to a Place, whereas Be-Obstructed applies to an object.

Neighboring Concepts:

Similar to...But different because...
Be-Blocked Be-Blocked applies to Places, whereas Be-Obstructed applies to objects.
Be-Closed Be-Closed applies to Portal-Coverings.
Be-Concealed Be-Concealed is specifically for objects that are concealed from view.
