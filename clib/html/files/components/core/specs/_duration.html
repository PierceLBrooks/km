Property: duration

How to represent properties

From: Event

To: Duration-Value

Description:

duration of an Event of Time-Interval

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Duration

  • 12 year old war
    number: 12
    unit of measure: year

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: forever, brief
    forever-brief-scale: forever..brief

  • brief moment
    value: brief
    relative to: Moment
