Concept: Follow

Superconcept: Locomotion

Component Semantics:

Follow
Definition: to do a Locomotion with the same path as a "prior" Locomotion.

Required Relationships:

agent: the Entity that moves (follows).
Comment: Follow is a type of Locomotion. Therefore, the agent (the Entity that causes the move) is the same as the object (the thing being moved).
base: the Entity that is being followed by the agent.

Examples:

  1. John followed Jane to the restaurant.
    agent: John
    base: Jane
