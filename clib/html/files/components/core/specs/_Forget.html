Concept: Forget

Superconcept: Action

Definition: Information once known to the agent becomes forgotten

Required Relationships:

object: the Information that becomes forgotten
agent: the Tangible-Entity that forgets the information
Comment: Before the forget action occurs, it's assumed that the information is in a Be-Known state in which the agent is the experiencer. The forget action takes the information out of that state.

Examples:

  1. John forgot the test was today.
    object: the test was today
    agent: John
Comment: Before the forget action, John knew the test was today.

Neighboring Concepts:

Similar to...But different because...
Learn The Learn action is the opposite of Forget; it causes an entity to know information
