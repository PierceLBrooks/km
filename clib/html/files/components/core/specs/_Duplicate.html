Concept: Duplicate

Superconcept: Copy

Definition: Duplicate some object, thereby creating a new object which is identical to the original in its properties, parts, material, and content.

Required Relationships:

object: The thing that's duplicated, i.e. the "original"
result: The thing that's created, i.e. the "Duplicate"

Other Common Relationships:

agent: the entity that caused the Duplication
instrument: the tool or aid that was used
by-means-of: the event that caused the Duplication to occur

Examples:

  1. Mel's copy of the Statue of David was a perfect duplicate of the original.
    agent: Mel
    object: the Statue of David
    result: Mel's copy
  2. Using the new digital copier, Julie duplicated the document.
    object: the document
    agent: Julie
    instrument: the new digital copier
    3. Comment: The result of this Duplicate process is left unstated. Implicitly it is "a Duplicate of the document".

Neighboring Concepts:

Similar to...But different because...
Copy Copy is more general than Duplicate. It involves creating a result that's identical to the original to some ways.
