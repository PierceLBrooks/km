Concept: Be-Blocked

Superconcept: Be-Inaccessible

Definition: An object, which is a Place, is inaccessible. A place is put into this state by Block, and removed from this state by Unblock.

Required Relationships:

object: the Place that is blocked

Other Common Relationships:

instrument: the entity causing the blockage

Examples:

  1. The path is blocked by the fallen tree.
    object: the path
    instrument: the fallen tree

Neighboring Concepts:

Similar to...But different because...
Be-Concealed Be-Concealed applies to objects that are hidden from view.
Be-Closed Be-Closed applies to Portals which are blocked to objects moving through.
Be-Obstructed Be-Obstructed applies to objects that are restricted in movement.
