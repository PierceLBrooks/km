Property: slope

How to represent properties

From: Spatial-Entity

To: Unitless-Value

Description:

the inclination of a Spatial-Entity

Value Description:

Numeric Values:

a real number (a ratio with an assumed denominator 1); nu unit of measurement UoM-Unitless

  • the slope gains 1 meter for every 2 you walk.
    number: .5
    unit of measure: UoM-Unitless

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: horizontal, gradual, steep, vertical
    *horizontal-vertical-scale: horizontal..gradual..steep..vertical

  • El Capitan rises vertically in front of your eyes.
    value: vertical
    relative to: ground slope
