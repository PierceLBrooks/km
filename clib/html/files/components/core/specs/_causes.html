Relation: causes

From: Event

To: Event

Description:

an Event causes another Event.

Examples:

  1. Splitting the atomic nucleus causes a release of energy.
  2. Timothy tipped the glass and it spilled all over the floor.
  3. The rise in room temperature caused the air-conditioning system to increase the airflow.

Neighboring Concepts:

Similar to...But different because...
agent An agent is an active Entity; it is typically responsible for causing an Event.
instrument An instrument is passive Entity; it is used by an Entity in an Event.
by-means-of by-means-of is usually an Event that is instrumental in causing another Event, but is not the cause of the second Event.
