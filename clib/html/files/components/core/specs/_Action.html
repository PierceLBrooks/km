Concept: Action

Superconcept: Event

Component Semantics:

Event
Definition: Things that cause change.

Required Relationships:

object: the Entity having the Action done to it

Other Common Relationships:

agent: the Entity that does the Action, or is responsible for it
instrument: the Entity used by the agent to do the Action
This is the top level concept for Events that have to do with doing or changing. Actions may be purely active (like Locomotion), purely passive (like Fall) or something in between (like the intransitive Break). Actions often correspond to changes in State.
