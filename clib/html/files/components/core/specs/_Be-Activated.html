Concept: Be-Activated

Superconcept: Be-Accessible

Definition: An object becomes activated. This is a specific type of accessibility. An object is put in this state by Activate.

Required Relationships:

object: the thing that becomes activated

Examples:

  1. Sally turned on the computer.
    object: the computer
    2. Comment: "Turned on" here would be represented by an Activate action, where the activation of the computer would be a result.
  2. Jerome armed the motion sensor.
    object: the motion sensor
    3. Comment: Activation is distinct from Operation. The motion sensor can now be used; triggering the motion sensor would not be an activate, since it is already activated.
