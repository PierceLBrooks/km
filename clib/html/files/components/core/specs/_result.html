Relation: result

From: Event

To: Entity

Description:

the entity that comes into existence as a result of an action.

Examples:

  1. Photosynthesis produces carbohydrates from carbon dioxide and water.
  2. Alexander Graham Bell invented the telephone.
  3. Internal combustion produces carbon monoxide.
  4. The angry mob broke through the wall creating a huge path for the trucks that followed.
