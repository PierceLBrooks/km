Concept: Gesticulate

Superconcept: Gesticulate

Definition: The physical act of making a gesture using a body part (e.g. pointing, looking) or using an object (e.g. a pointer).

Required Relationships:

object: Message being converted to sound.
result: the Sound that embodies the message

Examples:

  1. During the meeting, Fred gestured toward the whiteboard with a nod.
    agent: Fred
    instrument: Fred's head
    object: a Message that `the whiteboard is currently relevant'
    result: a pattern of Light acting as a Signal
