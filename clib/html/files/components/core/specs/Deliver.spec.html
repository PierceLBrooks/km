Concept: Deliver

Superconcept: Transfer

Definition: Change of possession of an object to a recipient by an agent who is not the donor

Required Relationships:

object: the thing that is delivered
recipient: the entity that possesses the object after the delivery

Other Common Relationships:

agent: the entity that causes the delivery. The agent is neither the donor nor the recipient, i.e. the delivery is by a third party
donor: the entity that possesses the object before the delivery
instrument: the medium of delivery of the object

Examples:

  1. FedEx delivered John's package to Sally.
    object: package
    agent: FedEx
    donor: John
    recipient: Sally

Neighboring Concepts:

Similar to...But different because...
Lose Lose is a special type of Transfer in which a donor is mandatory
Receive Receive is a special type of Transfer in which a recipient is mandatory
Withdraw Withdraw is a special type of Transfer in which the object is withdrawn from a base, thereby removing it from a Be-Stored state
