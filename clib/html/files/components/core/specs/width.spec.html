Property: width

How to represent properties

From: Spatial-Entity

To: Length-Value

Description:

width is a property for describing how big the horizontal measurement taken at right angles to the length is

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Length

  • 6 feet wide
    number: 6
    unit of measure: feet

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: narrow, wide
    *narrow-wide-scale: narow..wide

  • narrow road.
    value: narrowtall
    relative to: Road
