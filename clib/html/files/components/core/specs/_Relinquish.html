Concept: Relinquish

Superconcept: Lose

Definition: Intentional change of possession of an object by the donor

Required Relationships:

object: the thing that is relinquished
agent: the entity that relinquishes possession of the object. The agent is also the donor

Other Common Relationships:

recipient: the entity that possesses the object after the action
caused-by: the action that caused the loss

Examples:

  1. John relinquished his dog because of the attack.
    agent: John
    Comment: The agent is also the donor
    object: dog
    caused-by: the attack

Neighboring Concepts:

Similar to...But different because...
Lose Relinquish is a special type of Lose in which the action is intentional
Give Give is a special type of Relinquish which always involves a recipient
