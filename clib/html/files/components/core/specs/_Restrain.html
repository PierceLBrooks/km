Concept: Restrain

Superconcept: Obstruct

Definition: The object of a Restrain is prevented from moving. This action results in the object being in a Be-Restrained state, which can be undone by the Unrestrain action.

Required Relationships:

object: The thing that's restrained. The object is put in a Be-Restrained state by this action.

Other Common Relationships:

agent: The Tangible-Entity that's restraining the object
site: the place where the object is restrained

Examples:

  1. The wrestler restrained his opponent on the mat.
    object: the opponent
    agent: the wrestler
    site: the mat

Neighboring Concepts:

Similar to...But different because...
Block Prevent things from moving through some place, or along some object, by blocking such movements.
Confine Confine is more specific than Obstruct. It prevents the object from moving out of its enclosure.
Hold Hold is more specific than Obstruct. The object that's held can only be moved by the thing holding it.
Shut-Out Shut-Out is more specific than Obstruct. It prevents the object from moving into an enclosure.
Support Support is more specific than Obstruct. It prevents the object from falling.
