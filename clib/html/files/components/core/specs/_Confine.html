Concept: Confine

Superconcept: Obstruct

Definition: Prevent an object from being moved out of its enclosure. This action results in the object being in a Be-Confined state, which can be undone by Release'ing the object

Required Relationships:

object: The thing that's confined. The object is put in a Be-Confined state by this action.
base: The Tangible-Entity, playing the role of Container, in which the object is confined

Other Common Relationships:

agent: the entity that causes the confinement
location: the place inside the enclosure where the object is confined

Examples:

  1. The jailer confined the prisoner to his cell.
    object: the prisoner
    base: cell
    agent: the jailer

Neighboring Concepts:

Similar to...But different because...
Block Prevent things from moving through some place, or along some object, by blocking such movements.
Hold Hold is more specific than Obstruct. The object that's held can only be moved by the thing holding it.
Restrain Restrain is more specific than Obstruct. It prevents the object from moving at all.
Shut-Out Shut-Out is more specific than Obstruct. It prevents the object from moving into an enclosure.
Support Support is more specific than Obstruct. It prevents the object from falling.
