Concept: Activate

Definition: Some objects need to be activated before they can participate in an action. This action puts an object into a Be-Activated state.

Required Relationships:

object: The Entity that's activated

Other Common Relationships:

agent: the entity that activates the object
instrument: the instrument or tool used to activate the object
by-means-of: the event that is instrumental in activating the object

Examples:

  1. Roger activated the ignition system by turning the switch.
    object: the ignition system
    agent: Roger
    by-means-of: turning the switch

Neighboring Concepts:

Similar to...But different because...
Make-Accessible Activate is a type of Make-Accessible in which an object is put into a Be-Activated state.
Be-Activated Activate causes an object to enter a Be-Activated state.
Deactivate Deactivate is the opposite of Activate. It takes an object out of a Be-Activated state.
