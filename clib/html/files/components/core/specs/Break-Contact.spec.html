Concept: Break-Contact

Superconcept: Action

Definition: Break-Contact separates an object and a base which are touching.

Required Relationships:

object: The thing which is initially touching the base, i.e. they are in a Be-Touching state.
base: the thing which is initially touching the object.

Other Common Relationships:

agent: the entity that causes the object and the base to separate.

Examples:

  1. The chair was touching the wall until John separated them.
    object: the chair
    base: the wall
    agent: John
  2. The sharpshooter moved his finger off the trigger.
    agent: the sharpshooter
    object: the sharpshooter's finger
    base: the trigger

Neighboring Concepts:

Similar to...But different because...
Detach An object and a base may break contact if they are merely touching. In contrast, an object and a base must be attached for them to be detached.
