Concept: Mark

Superconcept: Embody

Superconcept: Produce

Definition: The act of embodying a message in the form of a printed ink, a physical mark or sketch, etc. This includes any action in which an object playing the role of writing implement (pen, pencil, printer, magic marker, spray paint can, finger dipped in paint, etc) writes, sketches, scribbles or sprays some kind of physical mark, set of markings, or written text.

Required Relationships:

object: The Message being embodied in ink, paint, etc.
result: The Tangible-Entity (ink, paint, etc.) that is left on the page, whiteboard.

Examples:

  1. Fred sketched the project schedule ghantt chart using ink on paper.
    agent: Fred
    object: scheduling information encoded in ghantt chart symbols
    result: a set of trails of ink markings laid out on paper

Neighboring Concepts:

Similar to...But different because...
Print Print is a more specific type of Mark in which the result is, by definition, a Physical-Document. With Mark, a Physical-Document *may* be the result, or it may be something less document-like (e.g. spray paint on a subway car).
Write The object of (or input to) a Write is an instance of Information whereas the object of a is a Message. Also, a results in an intangible Message, specifically a Document whereas results in a Physical-Mark.
