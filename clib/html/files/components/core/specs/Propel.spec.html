Concept: Propel

Superconcept: Move

Definition: An object Moves toward a Target.

Required Relationships:

object: the thing that's propelled toward the destination
destination: the Spatial-Entity, playing the role of a Target, to which the object is propelled

Other Common Relationships:

agent: the entity that caused the Propel to occur
instrument: the Entity used to propel the object
The other relationships used for Move can be used for Propel, too.

Examples:

  1. Tom shot a peanut to the squirrel with his sling shot.
    object: peanut
    destination: squirrel
    agent: Tom
    instrument: sling shot

Neighboring Concepts:

Similar to...But different because...
Move Propel is a type of Move in which the destination is a target
Slide Slide is a type of Move-Through with one extra property: throughout the move the object touches the path it's moving along
