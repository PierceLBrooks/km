Concept: Let-Go-Of

Superconcept: Unobstruct

Definition: Let go of an object that is being held.

Required Relationships:

object: Some tangible entity. Before this action, the object is assumed to be held by the agent, i.e. it's in a state of Be-Held. Afterwards it's no longer in that state.
agent: Somebody or something that's holding the object, then lets go of it

Other Common Relationships:

instrument: the thing (e.g. tool) used to let go of the object
by-means-of: the event that facilitates letting go of the object

Examples:

  1. John let the dog go.
    agent: John
    object: the dog
    2. Comment: It's assumed that John was holding the dog beforehand.

Neighboring Concepts:

Similar to...But different because...
Unobstruct Unobstruct is more general than Let-Go-Of. Unobstruct allows some object to move more freely, whereas Let-Go-Of (more specifically) is when an agent stops holding some object.
