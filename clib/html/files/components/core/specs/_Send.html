Concept: Send

Superconcept: Action

Definition: an intentional change of possession of an object from a donor to a recipient. It involves first a Relinquish of the object by the agent, followed by a Move of the object from the origin to the destination. Finally, the object is Delivered to the recipient.

Required Relationships:

agent: the entity that causes the Send action to occur
object: the thing that changes possession
recipient: the entity that possesses the object after the Send
donor: the entity that possesses the object after the Send

Subevents:

Relinquish: the object changes possession; afterwards the agent no longer possesses it
Move: the agent Moves the object from the location of the donor to the location of the recipient
Deliver: the object is delivered by the donor to the recipient

Subevent Order:

Relinquish then Move then Deliver

Other Common Relationships:

instrument: the medium of transfer of the object
origin: the location of the donor and, before the Send action occurs, the location of the object
path: the location of the object during the Move subevent
destination: the location of the recipient and, after the Send action is done, the location of the object

Examples:

  1. Victor is in Fresno. He sent a letter to Maurice in Detroit.
    agent: Victor
    donor: Victor
    object: a letter
    recipient: Maurice
    origin: Fresno
    destination: Detroit
  2. Marie sent the package to Jane using an express carrier.
    agent: Marie
    donor: Marie
    object: the package
    recipient: Jane
    instrument: an express carrier

Neighboring Concepts:

Similar to...But different because...
Transfer Transfer involves only a change of possession, while Send involves also a change in location of the object.
Move Move involves only a change in location, while Send involves also a change in possession of the object.
Deliver Deliver involves only a change of possession. The agent of the Deliver must be different than the donor.
