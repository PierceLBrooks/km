Relation: is-inside/encloses

From: Spatial-Entity

To: Spatial-Entity

Description:

One place or object is completely inside another.

Examples:

  1. The goalie fell inside the goal.
  2. The marble is inside the circle.

Neighboring Concepts:

Similar to...But different because...
content If an entity is the content of a container, then the entity is-inside the container. Therefore, the first statement subsumes the second. However, the first statement implies additional information. If an entity is the content of a container, not only is it inside (which is a structural assertion), it is also restricted from leaving and restricted from interacting with entities outside the container (which are functional assertions).
