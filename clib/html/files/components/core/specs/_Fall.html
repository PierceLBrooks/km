Concept: Fall

Superconcept: Move

Definition: An object falls toward earth.

Required Relationships:

object: the thing that falls
Comment: Note that fall is purely intransitive, i.e. it can never have an agent. However, Let-Fall, which causes a fall to occur, always has an agent.

Other Common Relationships:

origin: the initial location of the object before the fall
destination: the final location of the object after the fall
path: the Place which the object moves along during the fall. Typically, this is an object or surface along which the object falls.
toward: the location toward which the object falls. Unlike the destination, the object need not reach this Place, but it may.
away-from: the location that the object moves away from during the fall. Unlike the origin, the object need not start at this Place, but it may.

Examples:

  1. The anvil dropped from the 12th floor down the elevator shaft.
    object: the anvil
    origin: the 12th floor
    path: the elevator shaft
    2. Comment: Here "dropped" is translated to "fall".

Neighboring Concepts:

Similar to...But different because...
Move Fall is a special type of Move in which the direction is downward, toward the earth.
