Relation: beneficiary

From: Event

To: Entity

Description:

Entity benefits from Event.

Examples:

  1. Alfie won the lottery.
  2. The President declared and everyone is happy.
  3. I wrote a letter to prospective employers on Samir's behalf.

Neighboring Concepts:

Similar to...But different because...
experiencer An experiencer is the main passive participant in an event; a beneficiary of an Event benefits from the Event.
