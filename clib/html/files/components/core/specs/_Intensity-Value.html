Property: intensity/Intensity-Value

How to represent properties

From: Thing

To: Intensity-Value

Description:

the intensity of a Thing

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Intensity

  • the bulb that I bought has an intensity of 1000001 *kilocalorie/mole.
    number: 1000001
    unit of measure: *kilocalorie/mole

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: weak, strong
    *weak-strong-scale: weak..strong

  • The intensity of the newly discovered star is weak compared to that of the NorthStar.
    value: weak
    relative to: the instensity of NorthStar
