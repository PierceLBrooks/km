Concept: Learn

Superconcept: Interpret

Definition: An interpret in which the agent acquires new knowledge

Required Relationships:

object: the Information that's learned
agent: the Tangible-Entity that learns the information
Comment: This action puts the information into a Be-Known state in which the agent is the experiencer. If the information is already in this state, then the action cannot proceed (i.e. Learn requires that the agent is acquiring information that is new.)

Examples:

  1. Tom learned that elephants are big.
    agent: Tom
    object: that elephants are big

    2. Neighboring Concepts:

    Similar to...But different because...
    Teach The Teach action causes a recipient to learn something new, but unlike Learn, requires a Teacher.
    Forget The Forget action is the opposite of Learn; it causes an entity to forget Information
