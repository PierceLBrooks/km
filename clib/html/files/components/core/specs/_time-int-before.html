Relation: time-int-before

From: Time-Interval

To: Time-Interval

Description:

A time interval occurs before another. The first time interval ends before the second interval begins.

Examples:

  1. In an academic year, the thanksgiving break comes before the spring break.

Neighboring Concepts:

Similar to...But different because...
time-before The time-before relation can be applied to either time intervals or time instants. However, time-int-before applies only to time intervals
