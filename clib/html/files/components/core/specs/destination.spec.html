Relation: destination

From: Event

To: Spatial-Entity

Description:

When an entity moves, the destination is where it ends up.

Examples:

  1. John moved to Chicago.
  2. The hamburger slid into the trashbin. (destination=inside the trashbin)
  3. The storm moved toward the mountains, but ended up in the desert.

Neighboring Concepts:

Similar to...But different because...
toward toward indicates the place an event transpires toward, without actually ending there.
