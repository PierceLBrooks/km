Property: area/Area-Value

How to represent properties

From: Tangible-Entity

To: Area-Value

Description:

area is a property for describing how big the two-dimensional extent of a Tangible-Entity is

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Area

  • 573 square feet apartment
    number: 573
    unit of measure: square feet

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: tiny, huge
    *tiny-huge-scale: tiny..huge

  • Romania is large as a European country, but tiny compared to a US state.
    value: tiny
    relative to: US state

    value: large
    relative to: European country
