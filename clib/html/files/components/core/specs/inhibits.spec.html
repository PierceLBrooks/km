Relation: inhibits

From: Event

To: Event

Description:

an Event inhibits another Event but is insufficient to prevent it.

Examples:

  1. Obstructing an object inhibits moving it.
  2. Firing a warning shot inhibits the suspect from fleeing.

Neighboring Concepts:

Similar to...But different because...
prevents Prevents is stronger than inhibits: it asserts that the second event does not occur; an event that is merely inhibited may still occur.
