Concept: Unrestrain

Superconcept: Unobstruct

Definition: Allow some thing, which was previously restrained, to move freely

Required Relationships:

object: Some tangible entity. Before this action, the object is assumed to be completely restrained from moving, i.e. it's in a state of Be-Restrained. Afterwards it's no longer in that state.

Other Common Relationships:

agent: the entity that causes the restraints to be removed
instrument: the thing (e.g. tool) used to unrestrain the object
by-means-of: the event that facilitates removing the restraints from the object

Examples:

  1. By breaking his chain, Fido was free to run away.
    object: Fido
    agent: Fido
    by-means-of: breaking his chain
    2. Comment: It's assumed that Fido was restrained by the chain beforehand.

Neighboring Concepts:

Similar to...But different because...
Unobstruct Unobstruct is more general than Unrestrain. Unobstruct removes some obstacle, thereby allowing some object to move more freely. In contrast, Unrestrain (more specifically) removes all the restraints from an object, thereby allowing it complete freedom of movement.
