Concept: Make-Request

Superconcept: Communicate

Definition: An object (a Request) is requested by one entity of another usually in a polite manner.

Required Relationships:

agent: the entity making the request
object: the request being made
recipient: the entity the request is being made to

Other Common Relationships:

Examples:

  1. Larry requested of Sally that she close the door.
    agent: Larry
    object: to close the door
    recipient: Sally
