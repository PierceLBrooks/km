Concept: Unobstruct

Superconcept: Make-Accessible

Definition: Allow things that were previously obstructed to move freely.

Required Relationships:

object: The object that becomes unobstructed. It's assumed to be in a Be-Obstructed state before this action is taken, and afterwards it's no longer in that state.

Other Common Relationships:

agent: the entity that causes this action
instrument: the thing (e.g. tool) used to free the object
by-means-of: the event that frees the object

Examples:

  1. The policeman let the prisoner go.
    object: the prisoner
    agent: the policeman
    2. Comment: It's assumed that beforehand the policeman was obstucting the prisoner from moving in some way.
  2. The dog broke the leash that tied him.
    object: the dog
    agent: the dog
    by-means-of: breaking his leash
    3. Comment: The dog's movements become unobstructed.

Neighboring Concepts:

Similar to...But different because...
Make-Accessible Make-Accessible is more general than Unobstruct. Unobstruct applies only to the movement of things that were previously obstructed.
Open Open applies only to portal coverings, and the movement of things through portals.
Unblock Unblock makes a path accessible, whereas Unobstruct allows some thing to move freely.
