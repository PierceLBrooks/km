Property: height

How to represent properties

From: Spatial-Entity

To: Length-Value

Description:

height is a property for describing how big the vertical distance between the top and bottom of a Spatial-Entity is

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Length

  • 6 feet tall
    number: 6
    unit of measure: feet

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: short, long
    *short-long-scale: short..long

  • tall house but short building.
    value: tall
    relative to: House

    value: short
    relative to: Building
