Property: mass

How to represent properties

From: Tangible-Entity

To: Mass-Value

Description:

the property of a body that causes it to have weight in a gravitational field

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Mass

  • 12 kilogram bag

    unit of measure: kilogram

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: light, hefty

  • light laptop.
    value: light
    relative to: Laptop
