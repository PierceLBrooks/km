Concept: Expose

Superconcept: Make-Accessible

Definition: Expose some entity which was previously concealed (i.e. physically or visibly hidden).

Required Relationships:

object: The thing that's exposed. It's assumed to be in a Be-Concealed state before this action is taken, and afterwards it's no longer in that state.

Other Common Relationships:

agent: the entity that caused this action
base: the thing (entity or event) that the object becomes exposed to
by-means-of: the event that exposed the object

Examples:

  1. The plant was exposed to sunlight.
    object: the plant
    base: sunlight
    2. Comment: The plant is assumed to be concealed from sunlight beforehand.
  2. By removing the front panel, John revealed the broken spring.
    agent: John
    object: the broken spring
    by-means-of: removing the front panel
