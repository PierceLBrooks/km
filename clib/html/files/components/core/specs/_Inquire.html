Concept: Inquire

Superconcept: Express

Definition: Expression of a Question.

Required Relationships:

object: The agent expresses Information that is a Question.
agent: The agent expresses the Information in the object .
recipient: the entity that the inquire is expressed to.

Examples:

  1. Mick inquired to his brother, "Is it raining?".
    agent: Mick
    object: "Is it raining?"
    recipient: Mick's brother

Neighboring Concepts:

Similar to...But different because...
Interpret In a Interpret the agent is the passive receiver of Information, but in an Inquire the agent is actively expressing the Information which is playing the role of a Question.
Command In a Command the object plays the role of Instruction, but in an Inquire the information plays the role of a Question.
