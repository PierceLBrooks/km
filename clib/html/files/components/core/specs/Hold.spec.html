Concept: Hold

Superconcept: Obstruct

Definition: The object of a Hold can only be moved by the agent of the Hold. All other movements of the object are prevented. This action results in the object being in a Be-Held state, which can be undone by the Let-Go-Of action.

Required Relationships:

object: The thing that's held. The object is put in a Be-Held state by this action.
agent: The Tangible-Entity that's holding the object. Only this agent can move the object.

Other Common Relationships:

site: the place where the object is held.

Examples:

  1. The bird held the worm in his beak.
    object: the worm
    agent: the bird
    site: beak

Neighboring Concepts:

Similar to...But different because...
Block Prevent things from moving through some place, or along some object, by blocking such movements.
Confine Confine is more specific than Obstruct. It prevents the object from moving out of its enclosure.
Restrain Restrain is more specific than Obstruct. It prevents the object from moving at all.
Shut-Out Shut-Out is more specific than Obstruct. It prevents the object from moving into an enclosure.
Support Support is more specific than Obstruct. It prevents the object from falling.
