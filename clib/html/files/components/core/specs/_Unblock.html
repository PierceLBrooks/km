Concept: Unblock

Superconcept: Make-Accessible

Definition: Allow things to move through some place, or along some object, which was previously blocked.

Required Relationships:

object: The object or place that becomes unblocked. It's assumed to be in a Be-Blocked state before this action is taken, and afterwards it's no longer in that state.

Other Common Relationships:

agent: the entity that causes this action
instrument: the thing (e.g. tool) used to unblock the object.
The slots used to describe Make-Accessible actions in general, can be used here, too. For example, destination and by-means-of.

Examples:

  1. The policeman removed the barricade on the sidewalk.
    object: the sidewalk
    agent: the policeman
    by-means-of: move the barricade
    2. Comment: The sidewalk is the object that becomes unblocked by this action; it becomes accessible to things moving along it, such as pedestrians. Also note that "remove" here means "move".
  2. When the rubble was removed, the passageway became free.
    object: the passageway
    by-means-of: removal of the rubble
    3. Comment: The passageway (a place) became unblocked, so things were able to move through it.

Neighboring Concepts:

Similar to...But different because...
Make-Accessible Make-Accessible is more general than Unblock. Unblock applies only to the movement of things through places or along objects.
Open Open applies only to portal coverings, and the movement of things through portals.
Unobstruct Unobstruct takes an obstructed object and allows it to move.
