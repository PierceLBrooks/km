Concept: Be-Known

Superconcept: State

Definition: The state of Information in which it is known by an entity. An object is put in this state by Interpret. An object is taken out of this state by Forget.

Required Relationships:

object: the Information that becomes known
experiencer: the entity to which the object is known

Examples:

  1. After weeks of study, John knew the algorithms.
    object: The Information for which the information-content is "the algorithms"
    experiencer: John

Neighboring Concepts:

Similar to...But different because...
Be-Accessible Be-Accessible makes some tangible thing available, whereas Be-Known makes some information available.
