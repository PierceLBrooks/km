Concept: Give

Superconcept: Relinquish

Definition: A donor relinquishes an object to a recipient

Required Relationships:

object: the thing that is relinguished
agent: the entity that relinquishes the object. The agent is also the donor
recipient: the entity that possesses the object after the action

Other Common Relationships:

instrument: the medium through which the object is transferred

Examples:

  1. John gave his fiance a ring.
    agent: John
    Comment: The agent is also the donor
    recipient: John's fiance
    object: ring

Neighboring Concepts:

Similar to...But different because...
Donate Donate is a special type of Give in which the object is beneficial or valuable
Supply Supply is a special type of Give in which the object is made available, with the intention that others take it.
