Property: worth

How to represent properties

From: Entity

To: Worth-Value

Description:

an indefinite quantity of something having a specified value

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Worth

  • 10 dollars worth of gas
    number: 10
    unit of measure: *usdollar

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: cheap, inexpensive, expensive
    *cheap-expensive-scale: cheap..inexpensive..expensive

  • cheap date
    value: cheap
    relative to: Date
