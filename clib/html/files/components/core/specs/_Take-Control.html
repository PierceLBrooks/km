Concept: Take-Control

Superconcept: Action

Component Semantics:

Take-Control
Definition: change of control of an object from one agent to another
Required slot: agent
Restriction: causes object to Be-Controlled by the agent
Required slot: object
Restriction: is no longer controlled by whoever was controlling it; is now controlled by the agent
Required slot: postconditions
Restriction: object is no longer in any previous Be-Controlled state by other agents
Required slot: postconditions
Restriction: object is now in a Be-Controlled state by the agent
WordNet 1.6 Synsets: control#1
WordNet 1.6 Synsets: command#5
