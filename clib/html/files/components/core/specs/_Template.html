Concept: Template

Superconcept: Role

Definition: Template is the Role played by an Entity which is the object of a Copy.

Required Relationships:

played-by: the entity that's playing the role of a Template
in-event: the Copy event in which the entity is a Template

Examples:

  1. John copied Mary's notes.
    Comment: In this example, the Template role is described with these properties:
    in-event: Copy with agent=John
    played-by: Mary's notes
