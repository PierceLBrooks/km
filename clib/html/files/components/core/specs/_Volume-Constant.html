Property: volume/Volume-Value

How to represent properties

From: Tangible-Entity

To: Volume-Value

Description:

volume is a property for describing the amount of space occupied by a three-dimensional object as measured in cubic units

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Volume

  • 2 liter bottle
    number: 2
    unit of measure: liter

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: small, large
    *small-large-scale: small..large

  • This car's trunk is small for a full-size car.
    value: small
    relative to: the trunk of a full-size car

    • Ways to Determine a Value:

    To Determine a Value for a...Method
    Mixture The volume of a mixture is the sum of the volumes of the parts mixed. See Add-Volumes
