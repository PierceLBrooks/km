Relation: donor

From: Event

To: Entity

Description:

Entity releases object of Event (possibly unintentionally).

Examples:

  1. Edward sent the money.
  2. The Hewlett Foundation donated $400 million dollars to Stanford University.
  3. Edsger lost his marbles.
