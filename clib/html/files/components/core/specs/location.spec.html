Relation: location

From: Thing

To: Place

Description:

the Place where an Entity is located.

Examples:

  1. John's socks are where he left them.

Neighboring Concepts:

Similar to...But different because...
is-at is-at (and all of the other relations to do with space) allow you to specify the placement of an entity relative to other entities; location must refer to the abstract concept of a place.
site site is used to specify the location of an Event (as opposed to an Entity)
