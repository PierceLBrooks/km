Concept: Be-Unblocked

Superconcept: Be-Accessible

Definition: An object, which is a Place, is accessible for movement through or across it. An object is put in this state by Unblock. An object is taken out of this state by Block.

Required Relationships:

object: the Place that is accessible

Examples:

  1. The street is open.
    object: the street

Neighboring Concepts:

Similar to...But different because...
Be-Closed Be-Closed is specifically for portals through barriers.
Be-Concealed Be-Concealed is specifically for objects that are concealed.
Be-Obstructed Be-Obstructed is specifically for objects that are obstructed from moving.
