Concept: Buy

Superconcept: Obtain

Buy
Definition: Buy is an Obtain in which the recipient is the agent. A sum of money is exchanged.
Comment: Note that an instance of Buy doesn't imply that there is an instance of Sell. Each "sales transaction" is either one or the other, depending on the point of view. We don't have two instances.

Required Relationships:

donor: the one who relinquishes the object being bought

Other Common Relationships:

instrument: the sum of money used in the transaction

Examples:

  1. Jim bought my bike for $200.
    Comment: This is represented as a Buy with the following properties:
    agent: Jim
    donor: Jim
    instrument: $200

Neighboring Concepts:

Similar to...But different because...
Sell Sell is a Transfer in which the donor is the agent. A sum of money is exchanged.
