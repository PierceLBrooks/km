Relation: has-basic-functional-unit

From: Entity

To: Entity

Description:

the many identical things that combine functionally to make up a whole (as opposed to structurally).

Examples:

  1. A computer program is made up of many individual commands.
  2. Edgar's dissertation consists of two hundred arguments.
