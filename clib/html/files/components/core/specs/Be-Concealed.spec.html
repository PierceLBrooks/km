Concept: Be-Concealed

Superconcept: Be-Inaccessible

Definition: An object becomes physically or visibly hidden. This can describe the effects of numerous actions, as exemplified by the sentences below. An object is put in this state by Conceal. An object is taken out of this state by Expose.

Required Relationships:

object: the thing that becomes hidden

Other Common Relationships:

base: the entity that the object is concealed from.

Examples:

  1. Sally hides the jar of peanut butter.
    object: jar of peanut butter

Neighboring Concepts:

Similar to...But different because...
Be-Blocked When a Place is blocked, it is inaccessible.
