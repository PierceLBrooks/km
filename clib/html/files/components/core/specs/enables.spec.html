Relation: enables

From: Event

To: Event

Description:

an Event facilitates another Event, but is not sufficient to cause it.

Examples:

  1. Opening the door enables entry into the room.
  2. Building a new airport allowed larger planes to serve travelers.
  3. The hardware upgrade on the server enabled better performance.

Neighboring Concepts:

entails Entails is stronger: it asserts that if the first event occurs, the second event will necessarily occur.
