Concept: Supply

Superconcept: Relinquish

Definition: Supply is a type of Relinquish which enables the Replenishment of a Resource

Required Relationships:

object: the Tangible-Entity that is relinquished, and that serves to replenish a resource
agent: the Tangible-Entity that relinquishes the object. The agent is also the donor

Other Common Relationships:

recipient: the Tangible-Entity that receives the object when it is relinquished

Examples:

  1. The Indians supplied Lewis and Clark with food and tents.
    agent: Indians
    Comment: The agent is also the donor
    object: food and tents
    2. Comment: Descriptions of the food and tents, e.g. the events in which they serve as resources, can be specified as features of the Replenish action which this supply action enables

Neighboring Concepts:

Similar to...But different because...
Give In a give action, a donor relinquishes an object and the recipient takes possession of it. In contrast, in a supply action, the donor relinquishes an object in order to enable it to replenish a resource; the recipient does not necessarily gain possession, at least not immediately.
