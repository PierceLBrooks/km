Concept: Emit

Superconcept: Move

Definition: To send out Fluid, Particulate-Matter, or Energy.

Required Relationships:

object: the thing emitted; must be fluid, particulate-matter, or energy
origin: the location of the object before it's emitted. Must be either the location of the agent or the location of the instrument.

Other Common Relationships:

agent: the entity that causes the object to be emitted
instrument: the thing used to emit the object
by-means-of: the event that is instrumental in the emission
destination: the final location of the object after it's emitted
path: the Place which the object moves along after it's emitted. Typically, this is an object or surface along which the object moves
toward: the location toward which the object moves. Unlike the destination, the object need not reach this Place, but it may.
away-from: the location that the object moves away from. Unlike the origin, the object need not start at this Place, but it may.

Examples:

  1. The x-ray machine emits radiation into the patient.
    object: radiation
    instrument: the x-ray machine
    destination: a place inside the patient
    2. Comment: Here radiation is a form of energy.

Neighboring Concepts:

Similar to...But different because...
Move Emit is a special type of Move in which the object is fluid, particulate matter, or energy.
