Concept: Deactivate

Definition: Some objects can be prevented from participating in an action by deactivating them. This action takes an object out of a Be-Activated state.

Required Relationships:

object: The object that's Deactivated.

Other Common Relationships:

agent: the entity that deactivates the object
instrument: the instrument or tool used to deactivate the object
by-means-of: the event that is instrumental in deactivating the object

Examples:

  1. Simon deactivated the bomb with a plier.
    object: the bomb
    agent: Simon
    instrument: plier

Neighboring Concepts:

Similar to...But different because...
Make-Inaccessible Deactivate is a type of Make-Inaccessible in which an object is taken out of a Be-Activated state.
Be-Activated Deactivate causes an object to be removed from a Be-Activated state.
Activate Activate is the opposite of Deactivate. It puts an object into a Be-Activated state.
