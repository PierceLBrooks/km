Concept: Absorb

Superconcept: Take-In

Definition: Take-In a fluid into oneself.

Required Relationships:

object: the fluid that is taken in
agent: the Tangible-Entity, playing the role of a Container, that takes in the object. (Note: the agent is also the base of the Absorb.)

Other Common Relationships:

path: the Portal through which the object is taken in
origin: the initial location of the object before the Absorb. This must be a Spatial-Entity that is-outside the agent
destination: the final location of the object after the Absorb. This must be a Spatial-Entity that is-inside the agent
toward: the location toward which the object moves during the Absorb. Unlike the destination, the object need not reach this place, but it may
away-from: the location that the object moves away from during the Absorb. Unlike the origin, the object need not start at this place, but it may

Examples:

  1. The living sponge absorbs water full of nutrients.
    object: water full of nutrients
    agent: living sponge

Neighboring Concepts:

Similar to...But different because...
Enter Enter is a special type of Move-Into in which the object moves of its own volition, i.e. it locomotes
Load Load is a special type of Move-Into in which an object is inserted into a container, thereby activating it
