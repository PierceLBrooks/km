Concept: Be-Contained

Superconcept: State

Definition: An object is contained in a container. An object is put in this state by Move-Into. An object is taken out of this state by Move-Out-Of.

Required Relationships:

object: the thing that is contained in a container
base: the entity that contains the object.

Other Common Relationships:

Examples:

  1. The candy is in the jar.
    object: candy
    base: jar

Neighboring Concepts:

Similar to...But different because...
Be-Blocked Be-Blocked applies to Places which are inaccessible.
Be-Closed Be-Closed applies to Portals which are blocked to objects moving through.
Be-Obstructed Be-Obstructed applies to objects that are restricted in movement.
