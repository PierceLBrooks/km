Concept: Release-Resource

Superconcept: Action

Component Semantics:

Release-Resource
Definition: to make some Entity, that is playing the Role of a Resource, available.

Required Relationships:

object: The Resource that's made available. It's put in a Be-Available state by this action.

Other Common Relationships:

agent: the Entity that caused this action
base: the Thing (Entity or Event) that the object becomes available to.

Neighboring Concepts:

Similar to...But different because...
Release In Release-Resource, the "release" signifies the "becoming available" of the resource, while in Release, it signifies the freeing of something from confinement.
