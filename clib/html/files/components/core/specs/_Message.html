Concept: Message

Superconcept: Intangible-Entity

Definition: An encoding of information in a language

Required Relationships:

information-content: knowledge being communicated

Other Common Relationships:

base: The Language in which the information-content is encoded

Examples:

  1. The message in English was about the outcome of the soccer match.
    Comment: In this example, the Message is described by these properties:
    information-content: the outcome of the soccer match
    base: English
