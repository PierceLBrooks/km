Concept: Release

Superconcept: Unobstruct

Definition: Allow some thing to leave an enclosure or container.

Required Relationships:

object: Some tangible entity. Before this action, the object is assumed to be confined by the container, i.e. it's in a state of Be-Confined. Afterwards it's no longer in that state.
base: A container that's confining the object.

Other Common Relationships:

agent: the entity that causes the release
instrument: the thing (e.g. tool) used to release the object
by-means-of: the event that facilitates releasing the object

Examples:

  1. John released Fido from the pen with wire cutters.
    base: the pen
    object: Fido
    agent: John
    instrument: wire cutters
    2. Comment: It's assumed that Fido was confined to the pen beforehand.

Neighboring Concepts:

Similar to...But different because...
Detach Detach removes an attachment between two objects, whereas Release frees an object from its confinement in a container.
Take-Apart Take-Apart disassembles an object by removing all of its parts, whereas Release frees an object from its confinement in a container.
Break-Contact Break-Contact terminates the contact between two objects.
Unobstruct Unobstruct is more general than Release. Unobstruct allows some object to move more freely, whereas Release (more specifically) frees an object from its confinement in a container.
Move-Out-Of Release allows an object to move out of a container, whereas Move-Out-Of is the act of actually doing it.
