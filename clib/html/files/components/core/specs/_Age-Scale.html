Property: age

How to represent properties

From: Entity

To: Duration-Value

Description:

age is a property for describing how long an Entity has existed.

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Duration

  • 12 year old Scotch
    number: 12
    unit of measure: year

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: young, adolescent, juvenile, teenage, middle-aged, old
    *young-old-scale: young..adolescent..juvenile..teenage..middle-aged..old

  • She's a young woman, but she's old for a gymnast.
    value: young
    relative to: Woman

    value: old
    relative to: Gymnast
