Relation: instrument

From: Event

To: Entity

Description:

the entity that is used (by the agent if there is one) to perform an event.

Examples:

  1. John swatted the fly with a newspaper.
  2. Sally read the newspaper with a magnifying glass.
  3. The baseball broke the windshield.

Neighboring Concepts:

Similar to...But different because...
agent An agent is active, while an instrument is passive, being used by the agent (if there is one). An agent is typically considered sentient, if only metaphorically, while an instrument is not.
