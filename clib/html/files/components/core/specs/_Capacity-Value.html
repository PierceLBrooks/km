Property: capacity/Capacity-Value

How to represent properties

From: Tangible-Entity

To: Capacity-Value

Description:

the capacity (potential for storing or accomodating) of a Tangible-Entity

Value Description:

Relative Values:

a relative value along with a reference concept (what is the relative value relative to?)

values: none, roomy, capacious, unlimited
*none-unlimited-scale: none..roomy..capacious..unlimited

  • My car is roomy for a compact car.
    value: roomy
    relative to: a compact car
