Concept: Create

Superconcept: Action

Definition: Create something, i.e. cause it to come into existence.

Required Relationships:

result: The thing that's created.

Other Common Relationships:

agent: the entity that caused the creation.
instrument: the tool or aid that was used to create the result.
by-means-of: the event that caused the creation.

Examples:

  1. The little pig built a house of stone.
    result: a house of stone
    agent: the little pig
    2. Comment: You can represent "a house of stone" using the material slot.
  2. John sculpted a vase using a hammer and chisel.
    result: a vase
    agent: John
    instrument: hammer and chisel
  3. The reaction created hydrogen gas.
    result: hydrogen gas
    by-means-of: the reaction

Neighboring Concepts:

Similar to...But different because...
Produce Produce involves converting raw materials to order to create the result.
Copy Copy creates a result that resembles some original object.
