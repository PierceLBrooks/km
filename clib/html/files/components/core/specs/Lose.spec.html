Concept: Lose

Superconcept: Transfer

Definition: Change of possession of an object from a donor

Required Relationships:

object: the thing that is lost
donor: the entity that possesses the object before the loss, and which loses possession because of it

Other Common Relationships:

agent: the entity that causes the loss
recipient: the entity that possesses the object after the loss
caused-by: the action that caused the loss

Examples:

  1. John lost his wallet when he fell overboard.
    object: wallet
    donor: John
    caused-by: fall overboard

Neighboring Concepts:

Similar to...But different because...
Deliver Deliver is a special type of Transfer performed by an agent who is not the donor
Receive Receive is a special type of Transfer in which a recipient is mandatory
Withdraw Withdraw is a special type of Transfer in which the object is withdrawn from a base, thereby removing it from a Be-Stored state
