Concept: Record

Superconcept: Create

Definition: An action involving a Sensor such that a Recording is created as the result.

Required Relationships:

result: The thing that's recorded.
instrument: the tool or aid that was used in the record.

Other Common Relationships:

agent: the entity that caused the record.
by-means-of: the event that caused the record.

Other Common Relationships:

object: The (tangible) thing recorded, e.g. sound, light, brainwaves, etc.

Examples:

  1. Bob made an EEG recording of Fred's brainwaves.
    agent: Bob
    object: Fred's brainwaves
    instrument: EEG device
    result: EEG recording
  2. Pioneer 10's sensor recorded faint radio sources.
    agent: (none)
    object: faint radio sources
    instrument: sensor aboard Pioneer 10 spacecraft
    result: encoded sensor data

Neighboring Concepts:

Similar to...But different because...
Write The object of Write is Information which is intangible. By contrast, the object of Record is detectable by a sensor and thus tangible.
Sense Sense requires an experiencer but Record can happen autonomously.
