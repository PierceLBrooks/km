Concept: Conceal

Superconcept: Make-Inaccessible

Definition: Make an object concealed, possibly only with respect to some particular Entity or Event. This action results in the object being in a Be-Concealed state, which can be undone by performing an Expose action on the object.

Required Relationships:

object: the thing that's concealed; the object is put in a Be-Concealed state by this action

Other Common Relationships:

agent: the entity that causes the concealment
base: the entity or event from which the object is concealed

Examples:

  1. Sheilagh concealed her hideous tattoo
    object: her hideous tattoo.
    agent: Sheilagh
  2. Edwin concealed the engagement ring from his girlfriend.
    object: the engagement ring
    base: his girlfriend
    agent: Edwin
  3. The scalper's wares were concealed from observation.
    object: The scalper's wares
    base: observation

Neighboring Concepts:

Similar to...But different because...
Obstruct Obstruct affects the movement of something, but does not conceal it. Conceal doesn't necessarily prevent an object from being moved, or having any other particular action done to it.
Make-Inaccessible Conceal is more specific than Make-Inaccessible. Concealment is a particular kind of inaccessibility: inaccessible to view.
