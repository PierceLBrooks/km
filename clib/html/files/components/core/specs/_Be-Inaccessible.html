Concept: Be-Inaccessible

Superconcept: State

Definition: An object becomes inaccessible. An object is put in this state by Make-Inaccessible. An object is taken out of this state by Make-Accessible.

Required Relationships:

object: the thing that becomes inaccessible

Other Common Relationships:

base: the entity that the object is inaccessible to.

Examples:

  1. Sally closed the jar of peanut butter.
    object: peanut butter
  2. Jerome deactivated the database.
    object: the database
  3. King Arthur raised the drawbridge to the castle to keep the dragons out.
    object: the castle
    base: the dragons

Neighboring Concepts:

Similar to...But different because...
Be-Blocked Be-Blocked applies to Places which are inaccessible.
Be-Closed Be-Closed applies to Portals which are blocked to objects moving through.
Be-Concealed Be-Concealed applies to objects that are hidden from view.
Be-Obstructed Be-Obstructed applies to objects that are restricted in movement.
