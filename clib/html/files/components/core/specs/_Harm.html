Concept: Harm

Superconcept: Impair

Definition: Harm an object that is a living thing so it becomes weakened. The object can still perform its function, but at a diminished capacity.

Required Relationships:

object: The object that's harmed. The object is a living thing.

Other Common Relationships:

agent: the entity that caused the harm to occur.
instrument: the thing that was used to harm the object.
by-means-of: the event that caused the harm to occur.

Examples:

  1. The man harmed the dog by neglecting the dog.
    agent: the Man
    object: the Dog
    by-means-of: Neglect
    2. Comment: The dog is a living thing that becomes weakened or diminished.

Neighboring Concepts:

Similar to...But different because...
Impair An impair is a general kind of harm where the object does not have to be a living thing.
Damage A damage is similar to a harm, but the object being damaged is not a living thing.
