Concept: Resource

Superconcept: Role

Definition: A resource is the Role played by an entity which enables an event to occur. Some resources are Consumed by the event.

Required Relationships:

played-by: the Entity that's playing the role of Resource
in-event: the Event in which the entity is the Resource

Other Common Relationships:

instrument: frequently, this is the relationship between the event and the resource
raw-material: frequently, this is the relationship between the event and the resource

Examples:

  1. The fuel in my tank is a resource for my engine's combustion.
    Comment: In this example, the Resource role is described with these properties:
    in-event: my engine's combustion
    played-by: fuel in my tank

    Finally, the "fuel in my tank" is the raw-material of "my engine's combustion"
