Concept: Be-Stable

Superconcept: State

Definition: The state of an entity in which it's Stable, i.e. resistant to changes caused by some entity or some action.

Required Relationships:

object: the entity that's stable

Other Common Relationships:

base: the object resists changes caused by this entity or action
resulting-from: the Action that creates this stability
intensity: the degree of stability

Examples:

  1. The goalie was stable on his feet.
    object: the goalie
    base: Fall'ing
    intensity: *strong
  2. The tent became unstable when the pole broke.
    object: the tent
    resulting-from: breaking of the pole
    intensity: *weak
