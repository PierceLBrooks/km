Concept: Sell

Superconcept: Relinquish

Sell
Definition: Sell is a Transfer in which the donor is the agent. A sum of money is exchanged.
Comment: Note that an instance of Sell doesn't imply that there is an instance of Buy. Each "sales transaction" is either one or the other, depending on the point of view. We don't have two instances.

Required Relationships:

recipient: the one who received the object Relinquished

Other Common Relationships:

instrument: the sum of money used in the transaction

Examples:

  1. I sold Jim by bike for $200.
    Comment: This is represented as a Sell with the following properties:
    agent: Me
    recipient: Jim
    instrument: $200

Neighboring Concepts:

Similar to...But different because...
Buy Buy is an Obtain in which the recipient is the agent. A sum of money is exchanged.
