Relation: agent

From: Event

To: Entity

Description:

the entity that initiates, performs or causes an event. The `doer'.

Examples:

  1. John swatted the fly.
  2. The robber cracked open the safe.
  3. The president resumed the cold war.

Neighboring Concepts:

Similar to...But different because...
instrument An agent is active, while an instrument is passive. An agent is typically considered sentient, if only metaphorically, while an instrument need not be.
