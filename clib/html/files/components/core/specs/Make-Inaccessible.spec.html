Concept: Make-Inaccessible

Superconcept: Action

Definition: Make some entity inaccessible, perhaps to prevent it from being acted upon.

Required Relationships:

object: The thing that's made inaccessible. It's put in a Be-Inaccessible state by this action.

Other Common Relationships:

agent: the entity that caused this action
base: the thing (entity or event) that the object becomes inaccessible to
by-means-of: the event that caused the object to become inaccessible

Examples:

  1. John made the book unavailable for copying.
    agent: John
    object: the book
    base: copying
    2. Comment: The book may be available for other things, but not for copying.
  2. By moving it to the top shelf, Mom made the cookie jar inaccessible.
    agent: Mom
    object: the cookie jar
    by-means-of: moving the cookie jar to the top shelf

Neighboring Concepts:

Similar to...But different because...
Conceal Conceal is more specific than Make-Inaccessible. It hides an object from view.
Close Close applies only to portal coverings. It causes a portal to become inaccessible to things moving through it.
Block Block is more specific than Make-Inaccessible. It makes a path inaccessible to things moving through it or along it.
Obstruct Obstruct is more specific than Make-Inaccessible. It inhibits the movement of an object.
