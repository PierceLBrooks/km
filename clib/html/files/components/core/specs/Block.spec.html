Concept: Block

Superconcept: Make-Inaccessible

Definition: Prevent things from moving through some place, or along some object, by blocking such movements.

Required Relationships:

object: The object or place that becomes blocked. That object or place is put in a Be-Blocked state.

Other Common Relationships:

agent: the entity that causes this action
instrument: the thing (e.g. tool) used to block the object.
The slots used to describe Make-Inaccessible actions in general, can be used here, too. For example, base and by-means-of.

Examples:

  1. The policeman blocked the sidewalk with a barricade.
    object: the sidewalk
    agent: the policeman
    instrument: a barricade
    2. Comment: The sidewalk is the object that becomes blocked by this action; it becomes inaccessible to things moving along it, such as pedestrians.
  2. The rubble in the tunnel blocked the train's passage.
    object: the tunnel
    instrument: the rubble
    base: the train

Neighboring Concepts:

Similar to...But different because...
Make-Inaccessible Make-Inaccessible is more general than block. Block applies only to the movement of things through places or along objects.
Close Close applies only to portal coverings, and the movement of things through portals.
Obstruct Obstruct inhibits the movement of some object, whereas Block affects a pathway.
