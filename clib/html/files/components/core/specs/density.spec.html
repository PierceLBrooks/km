Property: density

How to represent properties

From: Tangible-Entity

To: Density-Value

Description:

the quantity per unit volume, unit area, or unit length

Value Description:

Numeric Values:

a number along with any of the units of measurement for UoM-Density

  • the density of water is 1kg/liter
    number: 1000
    unit of measure: gram/liter

    • Relative Values:

    a relative value along with a reference concept (what is the relative value relative to?)

    values: rare, dense
    *rare-dense: rare..dense

  • dense liquid
    value: dense
    relative to: Liquid
