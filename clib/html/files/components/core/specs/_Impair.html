Concept: Impair

Superconcept: Action

Definition: Impair an object so it becomes worse or less effective.

Required Relationships:

object: The object that's impaired.

Other Common Relationships:

agent: the entity that caused the impair to occur.
instrument: the thing that was used to impair the object.
by-means-of: the event that caused the impair to occur.

Examples:

  1. The fog impaired Billy's vision.
    agent: The Fog
    object: Billy's vision
    2. Comment: Billy's vision becomes less effective. Billy's vision, however, is not damaged. If its damaged, then use Damage to describe the action.

Neighboring Concepts:

Similar to...But different because...
Damage A damage is a special kind of impair in which the object is not a living thing.
Harm A harm is a special kind of impair in which the object is a living thing.
