Concept: Let-Fall

Superconcept: Unobstruct

Definition: Remove support from some object, causing it to fall.

Required Relationships:

object: Some tangible entity. Before this action, the object is assumed to be in a state of Be-Supported. Afterwards it's no longer in that state. Note that Let-Fall causes the object to fall.

Other Common Relationships:

agent: the entity that causes this action
instrument: the thing (e.g. tool) used to let the object fall
by-means-of: the event that lets the object fall

Examples:

  1. John yanked the table causing the dinnerware to fall.
    object: the dinnerware
    agent: John
    by-means-of: yanking the table
    2. Comment: Note that it's assumed that the table was supporting the dinnerware before the action occurs.

Neighboring Concepts:

Similar to...But different because...
Unobstruct Unobstruct is more general than Let-Fall. Unobstruct allows some object to move more freely, whereas Let-Fall (more specifically) removes support from some object, causing to to fall.
Fall Let-Fall causes an object to Fall.
