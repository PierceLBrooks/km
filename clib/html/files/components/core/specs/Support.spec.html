Concept: Support

Superconcept: Obstruct

Definition: Support puts the object into a Be-Supported state, which prevents it from Falling. Let-Fall takes the object out of the Be-Supported state.

Required Relationships:

object: The thing that's Supported. The object is put in a Be-Supported state by this action.

Other Common Relationships:

agent: the entity that causes the object to be supported
instrument: the thing used to support the object; "the support"

Examples:

  1. Tom supported the roof with a heavy beam..
    object: the roof
    agent: Tom
    instrument: heavy beam

Neighboring Concepts:

Similar to...But different because...
Block Prevent things from moving through some place, or along some object, by blocking such movements.
Confine Confine is more specific than Obstruct. It prevents the object from moving out of its enclosure.
Hold Hold is more specific than Obstruct. The object that's held can only be moved by the thing holding it.
Restrain Restrain is more specific than Obstruct. It prevents the object from moving at all.
Shut-Out Shut-Out is more specific than Obstruct. It prevents the object from moving into an enclosure.
